# چۆن ناوەرۆکێکی نوێ دروستبکەین
<!-- position: 2 -->

برۆنەژوورەوە لە **تەختەی بەڕێوەبەر**,کرتەیێک لە سەر **ناوەرۆکی نوێ** لە شریتی لاتەنیشتی ڕاست بکەن

1. سەردێڕێکی بنووسن.
2. ناوەرۆکیکی بۆ بابەتەکە بنووسن.
3. لەسەر دوگمەی **بڵاوکردن** کرتە بکەن.
4. تەواو!
